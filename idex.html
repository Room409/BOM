Que es Browser Object Model?

Browser Object MOdel o BOM es una interfaz de programacion que proporciona a los desarrolladores web acceso y control sobre el navegador web. Consta de un conjunto de objetos que representan diferentes componentes del navegador, como la ventana del anvegador, la barra de anvegacion, los frames/framesets, los historiales de navegacion, los objetos de cookies, etc.

Objeto window

El objeto "window" proporciona muchos otros métodos y propiedades útiles, como manipulación del historial de navegación (window.history), manejo de eventos (window.addEventListener), acceso al documento actual (window.document), manipulación de cookies (window.document.cookie), entre otros.

let currentURL = window.location.href;
console.log(currentURL); // Imprime la URL actual en la consola del navegador

window.open('https://www.ejemplo.com', '_blank');
Este código abrirá una nueva ventana o pestaña en el navegador con la URL especificada ('https://www.ejemplo.com').

document.

JavaScript te permite manipular y crear documentos HTML dinámicamente. Puedes utilizar JavaScript para seleccionar elementos existentes en el documento y modificar su contenido, estilo, atributos, entre otros. Además, también puedes crear nuevos elementos y agregarlos al documento.

let parrafo = document.getElementById("miParrafo"); parrafo.innerHTML = "Este es un nuevo contenido para el párrafo."; //Crea un nuevo parrafo desde js

let nuevoTitulo = document.createElement("h1"); nuevoTitulo.innerHTML = "Este es un nuevo título"; document.body.appendChild(nuevoTitulo) //Al ejecutar este código, se creará un nuevo título h1 con el contenido "Este es un nuevo título" y se agregará al final del cuerpo del documento.;

history

La API de Historial de JavaScript (History API) te permite manipular el historial del navegador, permitiéndote agregar y eliminar entradas, así como navegar hacia atrás y hacia adelante en el historial del navegador. Además, también puedes utilizar esta API para cambiar la URL en la barra de direcciones del navegador sin tener que cargar una página nueva.

history.length: devuelve el número de entradas en el historial del navegador.

history.back():navega a la página anterior en el historial del navegador.

history.forward():navega a la siguiente página en el historial del navegador.

history.go(n):navega a una entrada específica en el historial del navegador, donde n es el número de entradas que deseas saltar hacia adelante o hacia atrás (por ejemplo, history.go(-2) navegará dos entradas hacia atrás en el historial).

Location

El objeto location proporciona información sobre la URL actual del documento y te permite interactuar con ella. Puedes acceder a diferentes propiedades de location para obtener información sobre la URL, como el host, el protocolo, el path, los parámetros de búsqueda y más. Además, también puedes utilizar algunas funciones de location para redirigir o cargar nuevas URL.

location.href: devuelve la URL completa de la página actual

location.host: devuelve el nombre de dominio y el número de puerto de la página actual.

location.protocol: devuelve el protocolo de la URL actual (por ejemplo, "http:" o "https:").

location.pathname:devuelve la parte del path de la URL después del nombre de dominio.

location.search:devuelve los parámetros de búsqueda de la URL actual, incluyendo el signo de interrogación.

Navigation

El objeto navigation proporciona información y control sobre la navegación del usuario en la página web actual. Te permite acceder a diferentes propiedades y métodos para obtener información sobre la sesión de navegación del usuario, recargar la página, redirigir a otra página, entre otras acciones.

navigation.userAgent:devuelve una cadena que representa el agente de usuario del navegador.

navigation.appVersion:devuelve una cadena que representa la versión del navegador.

navigation.language: devuelve el idioma preferido del usuario.

navigation.onLine:indica si el navegador del usuario está en línea (conexión a Internet activa).

screen

El objeto screen proporciona información sobre las características de la pantalla del usuario, como la resolución, el color, la profundidad y el tamaño. Puedes acceder a diferentes propiedades de screen para obtener información sobre la pantalla del dispositivo del usuario y adaptar tu aplicación en consecuencia.

screen.width:devuelve el ancho en píxeles de la pantalla del dispositivo.

screen.height:devuelve la altura en píxeles de la pantalla del dispositivo.

screen.availWidth:devuelve el ancho en píxeles de la pantalla disponible para aplicaciones, teniendo en cuenta elementos como la barra de tareas o el dock.

screen.availHeight:devuelve la altura en píxeles de la pantalla disponible para aplicaciones, teniendo en cuenta elementos como la barra de tareas o el dock.

screen.colorDepth:devuelve la profundidad de color de la pantalla en bits por píxel.

screen.pixelDepth:devuelve la profundidad de píxel de la pantalla en bits por píxel.
